Sequence ADT

  • container of elements, indexed by set of contiguous non-negative integers
    • similar idea as vector
    • vectors usually have expectation of being implemented with a dynamic array
  • insert
    • inserts into specified position and shift elements to the right, increase container size by 1
  • append
    • similar to push_back
  • at
  • remove
    • removes at specified position, moving all elements one to the left, decreasing size by 1

Dictionary

  • collection of ordered pairs of the form (key, value)
  • Simple lookup table
    • adding (key, value) means that dict[key] should return value afterward
  • also called a map, or associative array
  • add
    • inserts pair to dictionary
  • overwrite
    • takes key and value, overwrites existing value
  • lookup takes key and returns associates value from dict
    • short form dict[key] == lookup(dict, key)
  • remove
  • print
    • assumed to not be sorted by default
  • implementation

C++ map datatype

map is an ordered (sorted) container, as are the related structures multimap, set, and multiset

  • iterating through map, get output in sorted order
  • map<T1, T2> m;
    • T1 must support < operator, strict weak ordering
    • anti-reflexive: doesn't go both ways
      • x is not less than itself
    • anti-symmetric:
      • if a < y, y !< a
    • transitive:
      • if a < b, b < c, a < c

for(map<string, int>::const_iterator i = m.begin(); i != m.end(); i++) {
	cout << (*i).first << " " << (*i).second << endl;
}

for (auto const& x : m) {
	cout << x.first << " " << x.second << endl;
}

Counting the number of words in an article

string trim (string s) {
	string ans = "";
	for (int i = 0; i < s.length(); i++) {
		const char c = s.at(i);
		if (isalpha(c)) {
			ans += tolower(c);
		}
	}
	return ans;
}

int main() {
	map<string,int> m;
	
	string token;
	while (cin >> token) {
		m[trim(token)]++;
	}

	for (auto const& c : m) {
		cout << c.first << " " << c.second << endl;
  }
